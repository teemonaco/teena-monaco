Scrapbook.com E-commerce Redesign

Improving Checkout Experience and Overall Site Usability

Overview

Led a comprehensive redesign of the Scrapbook.com e-commerce site...

  • Role: Lead UX/UI Designer & Front-End Developer
  • Duration: February 2023 - Present
  • Tools Used: Figma, Looker Studio, HTML, CSS, JavaScript, PHP, Google Analytics, Exponea, Google Search Console

Challenge

The existing e-commerce site had a high cart abandonment rate and frequent user complaints about the checkout process. The goal was to redesign the site to improve user experience, accessibility, and conversion rates, specifically targeting an older user demographic.

  • Design and functionality were inconsistent, leading to a disjointed user experience.
  • Error messages were unclear, causing confusion for users.
  • The site was not responsive, resulting in poor usability across different devices.
  • Lack of viewport settings led to significant layout issues on various screens.
  • Pages loaded in quirks mode, adversely affecting performance and display consistency.
  • Slow loading times significantly detracted from the user experience.
  • Failed to meet AA contrast standards, making text difficult to read.
  • Forms lacked accessibility features, hindering usability for all users.

Process

Research: Conducted user research including surveys, interviews, and usability testing to understand the needs and pain points of the target demographic. Analyzed data from Google Analytics, Exponea, and Google Search Console to track user behavior and identify key areas for improvement.

Design: Developed a comprehensive Accessible Design System following WCAG standards. Created wireframes, mock-ups, and high-fidelity designs in Figma and Adobe XD. Focused on making the interface intuitive and accessible, with clear navigation and readability.

Prototyping: Utilized rapid prototyping techniques to create both low-fidelity and high-fidelity prototypes. Conducted regular demos to stakeholders to gather feedback and iterate on designs.

Implementation: Worked closely with the development team to implement the new designs. Developed a custom framework dashboard for developers to streamline code implementation and ensure consistency. Optimized site performance by creating a custom Font Awesome kit, significantly reducing load times.

Solution

Redesigned the entire checkout process to streamline the user experience and reduce cart abandonment. Key steps included:

  • Integrated PayPal and Apple Pay to provide seamless and flexible payment options.
  • Improved overall site accessibility by implementing larger fonts, high-contrast themes, and simplified navigation.
  • Developed a custom Font Awesome kit to enhance performance and reduce load times.
  • Created a framework dashboard for developers to ensure consistent code implementation and streamline the development process.

Results

  • Reduced cart abandonment rate by 20%, resulting in a significant increase in completed purchases.
  • Enhanced user satisfaction, reflected in a 30% decrease in checkout-related complaints.
  • Increased accessibility compliance, receiving positive feedback from users with visual impairments.
  • Improved site performance, with a 15% faster load time due to the optimized Font Awesome kit.
  • Data-driven insights: Created custom dashboards in Looker Studio to continuously track user behavior and measure the impact of design changes, leading to more informed decision-making.
